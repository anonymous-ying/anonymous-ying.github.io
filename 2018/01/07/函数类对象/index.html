发表于 2018-01-07 | 分类于 Python
Python函数类对象笔记

函数


对多次反复调用的代码块进行封装

format

#定义
def FUNNAME(*args):
   CODE...
   [reutn value]
#调用
FUNNAME(*args)

参数

  • 位置参数
        依次传入，当函数有参数时，必需填入
#arg1, arg2为形式参数,函数内部使用
def FUNCNAME(arg1,arg2):
    print arg1, arg2
#a1,a2为实参，可以不与形参同名
FUNCNAME(a1, a2)
  • 关键字参数
        一般用于定义默认参数，key=value
def FUNCNAME(arg1=a1, args2):
    CODE
  • 可变参数：传入[不定个数]的参数组

    非关键字可变长参数(元组)
        使用：[ * ]，可以是任意数量的参数，再将参数按位置封装成元组传入

def FUNCNAME(*tuple_name):
    print tuple_name
FUNCNAME(a1,a2,a3,...)

    关键字可变长参数(字典)
        使用：[ ** ]，将任意数量的关键字参数封装成字典的格式传入，允许参数缺失，不用按照位置传参

def FUNCNAME(**dict_name):
    print dict_name
FUNCNAME(a1=v1, a2=v2, a3=v3,...)

返回值

return | yield：让函数中断执行，如果函数有执行结果，将执行结果返回，可以return None或者空，return下面的代码将不被执行。
    yield - 迭代器，上一章

def FUNCNAME(*ags):
    print "start run"
    return
    print "Not executed"
r = FUNCNAME()
r -> 空

变量作用域

顺序：局部变量 -> 全局变量
        全局变量
声明在函数/类外层，全局可以调用，但不能修改
global：让函数/类能够修改全局变量的值返回
        局部变量
声明在函数/类内部，由函数/类自身调用

匿名函数 - lambda

直接将函数执行结果返回，便捷形式函数,可以不定义func_name、return
format:
    func_name = lambda [*args]: CODE

递归回调函数

    递归函数：函数内部调用自身函数
        斐波序列、二分
    回调函数：函数作为[结果值]返回
        装饰器

类、对象


减少重复代码，加强扩展性，易读易改

  • 类：具有相同属性&方法的[对象]集合
  • 对象：类的其中一个实例

format

class ClassName(base):
    'class document string'
    attr = value
    #初始化实例参数(变量)
    def __init__(self, arg1, arg2):
        #将参数绑定到self(实例本身)
        self.arg1 = arg1
        self.arg2 = arg2
    def function(self):
        #引用实例参数(变量)
        print "%s : %s" (self.arg1, self.arg2)
    CODE...
#实例化类(对象)
obejct = ClassName(a1, a2)
object = ClassName(b1, b2)
#变量引用
ClassName.attr
object.attr
object.arg1
#对象方法调用
object.function()

参数

实例参数，在创建实例的时候通过构造函数(__init__)传参

  • 参数传入时绑定到self(实例对象自身)，通过self.key引用
  • 每个实例只能访问自己的参数/变量(通过self)

变量

  • 类变量
        在[类]中，在[函数体]外
        在这个类的所有[实例]之间共享，类成员都可以访问

  • 实例变量
        定义在[类函数体内]，绑定到self
        只作用于当前实例，只能由实例本身访问，类都不能访问

  • 私有变量/特殊变量(属性访问控制)
        私有变量：_attr = value
        特殊变量：__attr__ = value

类特点

封装

抽象得到的数据和行为(或功能)相结合 抽象类 -> 实例调用实例方法

  • 实例方法 def func(self)：绑定到self的方法
        通过实例对象访问：object.func()
            不用传参，参数在实例对象时由__init__绑定到self

  • 类方法 @classmethod：通过类名访问

@classmethod
def classfunc():
    CODE

        ClassName.classfunc()
        类方法不能访问实例变量

  • 静态方法 @staticmethod：
@staticmethod
def classfunc():
    CODE

        不能访问类变量跟实例变量，相当于虽然在类里，但是又独立出去

  • 属性方法 @propertymethod：
@propertymethod
def classfunc():
    CODE

        让方法变成以属性的方法访问便可以执行，不用()

继承

通过[继承]享有[父类的属性]进一步减少代码重用

class Father(object):
    def __init__(self, name, age):
        CODE
    CODE
class Son(Father):
    #重构参数
    def __init__(self, name, age, play):
        #继承父类参数
        super(Son, self).__init__(name, age)
        self.play = play
    CODE
  • 类参数：
        object[base]：所有类的基类[父类名]
  • 初始化参数
        不会自动继承父类_init__
            调用super(sonclass, self).    __init__(farg1,farg2)
  • 参数读取顺序：先在[本类]中查找，再到[父类]中查找
  • 多重继承：组建一个类列表当base

BIF(类方法)

  • issubclass(sub, (sup,..))：判断sub是否为sup(可以传入一个类元组)的子类
  • isinstance(obj, (class,…))：判断对象是否属于这个类
  • hasattr(obj, name)：判断对象是否有这个属性、方法
  • [value|func = ]getattr(obj, name)：获取对象的属性、方法
  • setattr(obj, name, value[func])：设置这个对象的新属性、方法
  • delattr(obj, name)：删除这个对象的一个属性、方法
    一般用于做关系映射
