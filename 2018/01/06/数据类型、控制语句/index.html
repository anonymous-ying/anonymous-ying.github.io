Python基础-运算符、占位符、数据类型、控制语句

操作符

  • 算术运算
        +、-、*、/
        %：取模(除法取余)
        **：求幂
        //：取整除(商的整数部分)
  • 赋值运算
        =
        +=、-=、=、/=、%=、*=、//=
  • 比较运算
        ==、!=
        >、>=、<、<=
  • 位运算/逻辑运算
        &&：与 - and
        ||：或 - or
        ^：非 - not
        -：反
        <<：左移
        >>：右移
  • 成员运算
        in
        not in
  • 身份运算
        is
        not is
  • 转义符
        \
            \n：换行
            \r：回车
            \t：制表

占位符


不能出现缺失值

  • % - 多个占位使用tuple进行代入-“%s:%s” % (v1,v2)
        %s：字符串
        %d：整数
        %f：小数
            %0.2f - 两位小数点
        ”%(key)[s|d|f]” % {‘k1’:’v1’,’k2’,v2,…}：键值占位
            s：字符串
            d：整数
            f：小数
            e.g：”%(name)s” % {‘name’:’string’}
  • format：{}
        ”{}:{}”.format(v1,v2)
        ”{k1}:{k2}”.format(k1=v1, k2=v2)

查看数据类型函数：type(value)

基本数据类型

  • int：有符号整型
  • float：浮点型
  • complex：复数
  • bool：布尔判断
        true
        false

字符串 str - “”

python的str为不可变对象，每次迭代都会生成新的str对象来存储字符串，字符串越长，创建的str对象越多占内存越大

  • 定义：string = “” | string = str()
  • r：r’str’ - 原字符输出，忽略转义符号
  • 字符串元组
        可索引
        不支持索引修改
        支持转换为list、tuple，转后不可以直接转回str，需要做遍历拼接字符(修改也是)
  • BIF(built-in function-内建函数)
        编码 - python内部使用unicode编码
            decode()：将某编码转为unicode
            encode()：将unicode转为某编码
        大小写
            str.upper()：将英文字母都转换为大写
            str.lower()：将英文字母都转换为小写
        字符串操作
            str.split([sign])：将字符串以sign分割为列表，列表不包括sign
                默认空格
            ’[sign]’.join(strlist)：用sign将字符串列表里的每个值拼成一个字符串
            str.replace(‘old’,’new’)：替换字符串字符
            str.[l|r]strip()：去掉字符串的空格
                不能去掉中间空格，l-前 | r-后 | 默认前后
            str.count(str[,start[,end]])：统计字符
            str.index(str[,start[,end]])：查看字符所在索引
        判断(bool)
            str.isupper()：是否都为大写字母
            str.islower()：是否都为小写字母
            str.isalpha()：是否只包含字母
            str.isdigit()：是否只包含数字
            str.isalnum()：是否只包含数字和字母
            str.startswith(str)：是否是str开头
            str.endswith(str)：是否是str结尾

列表 list - []

  • 定义：li = [] | li = list()
  • 多个变量的集合，可以存放不同的变量类型在同一个列表中，可以修改增删列表中的值
  • python2的列表中中文会被转码，索引取值还是会编码错误!!!在for中遍历时候不会出现编码问题
  • 索引 - 列表中的每个值都有相对应的下表(0开始)
        通过索引进行取值
            list[index]：获取指定下标的值
            正：前往后取 - list[0] 开始第一位
            负：后往前取 - list[-1] 倒数第一位
            list[[start]:[end]]：范围取值
                list[N:M]：从N截取到M，前包后不包
                list[:N]：从开始截取到索引为N
                list[N:]：从索引为N截取到最后
        通过索引进行替换
            list[N] = newValue：替换索引为N的值

  • BIF
        list.append(value)：往列表中添加新值
            一般配合for遍历添加列表
        list.insert(index, value)：指定索引插入新值
        list.index(value)：查看某个值的索引
        list.count(value)：统计某值的出现的次数
        list.extent(list)：合并两个列表为一个列表
            运算符
                list1 + list2：两个列表拼接
                list1 * 3：列表乘倍
        list.sort()：对列表进行从小到大排序
        list.reverse()：对列表进行反转
        list.pop()：取出一个值，并且在列表中删除此值，没参数默认从最后一个
        list.remove()：删除一个值

  • 一般配合使用函数
        del list[k]：删除一个列表元素
        列表成员判断(bool)：in | not in
        len(list)：列表长度
        enumrate(list[, start])：输出列表索引化

    for index,value enumrate(list, 1):
    print index, value

        zip(list1, list2)：聚合多个等长列表，将相同索引的值合并为元组

    for li1,li2 in zip (list1,list2):
    print li1, li2

元组 tuple - ()

  • 定义：tup = tuple(list)
  • 不可变列表，不能在定义后增删改该列表
  • 通过索引或遍历的方式取值
  • BIF
        tup.count()
        tup.index()

集合 set - {}

  • 去除重复的列表
  • 定义：s = set(list)
  • 无序可变
  • BIF
        set.add(value)：添加
        set.update(list)：合并
        set.remove(value)：删除
        set.pop()：出栈
  • 功能
        成员关系测试
            in / not in
        集合关联
            交集 - | ：两个列表合并输出
            并集 - & ：两个列表共同拥有的元素
            补集 - - ：两个列表中不共同的元素

字典 dict - {}

  • 定义：di = dict() | di = {[key:value]}
  • 输出所有键值时乱序
  • 键值对应的方式进行数据取值
        dict[key]
  • 添加数值
        dict[newkey] = newvalue
  • BIF
        dict.update(dict2)：合并两个字典
        dict.keys()：列表的形式返回字典的所有键
        dict.values()：列表的形式返回字典所有的值
        dict.items()：元组列表的形式返回字典的键值
        dict.has_key(key)：判断(bool)字典中是否存在该键
        dict.get(key)：获得字典中的指定键的值
        dict.pop()：返回一个键值并删除
        dict.clear()：删除所有键值

类型转换

  • int()
        可以将str类型的数字转为int型数字
        字母不能进行int()转换
  • str()
  • list()
  • tuple()
        将列表形式类型转换为元组
  • set()
        将列表形式类型转换为集合
  • iter()
        将列表形式类型转换为迭代器

控制语句


所有的语句皆可以进行套嵌，先结束内层循环后才会返回执行上一层循环,套嵌之后避免内存溢出必需要有结束条件

循环

关键字

  • break
        跳出整个循环循环,结束循环
  • continue
        跳过本次循环，回到判断条件
  • pass
        忽略本次循环，进入下次循环

for

  • 遍历列表元素
  • format
    for 临时变量 in 可迭代对象:
    CODE
  • 可迭代对象
        列表/元组/字典/集合(多个元素)
        len()/range()/enumrate()/zip()
  • 三元表达式
    li = [x for x in range(10)]

while(bool)

  • 根据条件循环
  • format

判断 - if(bool)

  • if
        format：
    if CONDITION:
    CODE
elif CONDITION:
    CODE
...
else:
    CODE
        多重条件:
            CONDITION1 and[or not(!)] CONDITION2
        三元表达式：
            format：
    CODE1 if CONDITION else CODE2
            如果CONDITION成立则执行CODE1，否则执行CODE2

异常处理(try)

  • format
    try:
    CODE
except [Exception]:
    CODE
except (exception1, exception2):
    CODE
except Exception as e:
    CODE
    print e
else:
    CODE
finally:
    CODE
    如果try里面的代码块出现异常(抛出运行错误信息)，就会执行except代码块内容，如果try整个代码块能够顺利运行就执行else内容(反之不执行)，finally是无论代码块有无异常在最后都会执行的代码块
  • Exception：无论出现何种异常(有特殊)，都会执行该代码块，as e为该异常的信息
  • [Exception]：针对特定异常来执行该代码块
                                     常见异常类型
AttributeError：属性错误
IOError：输入/输出错误，一般为无法打开文件
ImportError：模块导入错误，一般为路径、名称错误或模块未安装
IndexError：索引错误
KeyError：访问字典不存在的键
KeyboardInterrupt：键入Ctrl+C。不能由全局异常捕捉
EOFError：键入Ctrl+D
NameError：变量未被定义
IndentationError：语法错误，一般为代码缩进。不能由全局异常捕捉
SyntaxError：编译错误，代码语法错误。不能由全局异常捕捉
TypeError：传入对象类型与要求不符
AssertionError：断言触发异常

自定义异常 raise

  • 重写异常类
    class MyException(Exception):
    def __init__(self, msg):
        self.message = msg
    def __str__(self):
        return self.message
try:
    CODE
    raise MyException('this input exception message')
except MyExceotion as e:
    print e
    初始化异常时传入异常提示的信息，再通过__str__返回异常信息
  • raise：如果想出发自定义异常需要手动raise触发且传入该异常的错误信息
  • 涉及业务逻辑，需要自定义一些业务逻辑需要的异常

断言 assert

  • format：

    assert CONDITION

  • 用于明确判断前面代码块是否符号某个条件才能让程序继续执行，否则让程序抛出异常
