Python文件操作模块

发表于 2018-01-17 | 分类于 Python
python操作文件、文件路径、文件读写、压缩 - (sys、os、open、shutil、tarfile、zipfile)

sys


sys.argv[N]：程序运行的参数[列表]
    sys.argv[0] - 程序文件名
    sys.argv[1 -> N] - 程序接收的参数
sys.path：python模块的环境变量[列表]
    添加环境变量 - sys.path.append(PATH)

os - 操作文件夹\路径



绝对路径：总是从根开始(/)
相对路径：相对当前路径, . | ..(当前|上一级)


PATH

  • os.path.abspath(PATH)：返回相对当前文件夹的绝对路径
        一般配合PATH：
            __file__(内置变量) - 当前文件的名称
            sys.argv[0] - 运行脚本的第一个参数(程序名)
        一般配合sys.path添加自定义模块的路径，如下
#返回文件相对路径
print __file__
#返回文件绝对路进
print os.path.abspath(__file__)
#返回文件所在目录位置
print os.path.dirname(os.path.abspath(__file__))
base_dir = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
print base_dir
#添加父目录为自定义模块的环境变量
sys.path.append(base_dir)
print sys.path
  • os.path.dirname(PATH)：返回路径最后一个斜杠前面的路径(父目录的绝对路径)
  • os.path.basename(PATH)：返回路径最后一个斜杠后面的路径|文件(相对路径名|文件名)
  • os.path.join(dirname,basename)：路径拼接
  • os.path.split(PATH)：分隔最后一个斜杠，元组方式返回
  • os.path.exists(PATH|FILE)：判断路径|路径是否存在
  • os.path.isdir(PATH)：判断是否为路径
  • os.path.isfile(FILE)：判断是否为文件
  • os.getcwd()：返回当前所在位置的绝对路径
  • os.walk(PATH)：返回一个walk生成对象
        walk生成对象：元组(dirname,sondir,file)
            dirname：PATH的父目录(绝对路径)
            sondir：子目录列表，子目录会继续遍历，直到最后一层目录
            file：当前文件夹下的文件列表
for dirname,sondir,files in os.walk(os.walk):
    for name in files:
        print "files:",os.path.join(dirname, name)
    for name in sondir:
        print "sonfolder:",os.path.join(dirname, name)
    print "---------------------------"
  • os.chdir(PATH)：切换当前所在目录
  • os.listdir(PATH)：列出目录下的内容
  • os.makedirs(PATH[, mode=755])：创建文件夹
        open(file, ‘w’) - 创建文件
  • os.remove(FIEL)：删除文件
  • os.removedirs(PATH)：删除空文件夹
  • os.chmod(PATH,MODE)：修改文件夹权限
  • os.chown(PATH,UID,GID)：修改文件所属用户/组

open() - 读写文件


format：open(file[,mode[,buffering]])

模式

  • r：以读的方式打开文件(默认),文件事先存在，不支持写
  • w：以写的方式打开文件(下次写覆盖),文件不存在时自动创建，不支持读
  • a：追加模式写(不覆盖原有内容)，文件不存在时自动创建，不支持读
  • b：以二进制的格式写入内容(rb,wb,ab)
  • +：读/写模式，让打开模式同时支持读写
        r+：读写模式打开文件，文件指针放在文件开头，不覆盖原有内容
        w+：读写模式打开文件，文件指针放在文件开头，覆盖原有内容
        a+：读写模式打开文件，如果文件存在，文件指针放在文件结尾，如果文件不存在，创建新文件读写

        
返回值：文件对象

方法

读

  • fobj.read([size])：读取文件，默认一次读取全部，或者指定读取到某个位置(指针)
  • fobj.tell()：获取文件当前指针位置
  • fobj.seek(index)：调整文件指针
  • fobj.readline()：读取文件一行内容(识别\n)，文件指针指向下一行，返回值 - 字符串
  • fobj.readlines()：列表的形式记录没一行,指针指向文件末尾，返回值 - 列表
  • fobj.xreadlines()：迭代输出没一行

写

  • fobj.write(str)：往文件写入内容
        \n：换行
  • fobj.writelines(str)：往文件写入一行内容

关闭

  • fobj.close()
  • with open(file[, mode]) as f:

shutil - 操作文件

复制

  • shutil.copyfile(src, dst)：复制文件内容，不包括文件权限状态
  • shutil.copymode(src, dst)：仅复制文件权限
  • shutil.copystat(src, dst)：仅复制所有的状态信息
  • shutil.copy(src, dst)：复制文件内容及权限  [copyfile+copymode]
  • shutil.copy2(src, dst)：复制文件内容及所有状态信息  [copyfile+copystat]
  • shutil.copytree(src, dst, symlinks=False, ignore=None)：递归复制文件内容及状态信息

移动/重命名(mv)

  • shutil.move(src, dst)
        文件存在的路径没有改变的情况下为改名  move(oldname, newname)
        文件路径改变的情况下为移动  move(source, destination[newname])

删除

  • shutil.rmtree(path)：删除文件夹(可以删除不空文件夹)
    [os.remove(file) | os.removedirs(path)]

压缩、解压

tarfile(.tar\.tar.gz\.tgz\.tar.bz2)

format：tarfile.open(name=None, mode=’r’)
mode：
    读：r、r:*、r:[gz\bz2]
    写：w、w:、w:[gz\bz2]

创建压缩、解压对象(tar = tarfile.open(file,’r|w’))
压缩文件 - 可以是文件夹
    tar.add(FILE|PAHT)

#处理多文件
for dirname,sondir,files in os.walk(PATH):
    for f in files:
        fullpath = os.path.join(dirname, f)
        tar.add(fullpath)

解压文件
    tar.list() #查看压缩包下的文件
    tar.extractall(path=’.’) #解压所有文件，默认解压在当前文件夹
关闭对象：tar.close()

zipfile(.zip)

  • zipobj = zipfile.ZipFile(file, mode=’r’)：创建压缩解压对象
        mode：r - read | w - write | a - append
  • zipobj.write(file, compress_type=zipfile.ZIP_DEFLATED)：往压缩包添加文件
        只能是文件，如果是文件夹，文件夹下的文件将不被压缩(可以通过os.walk遍历，截取上级目录拼接文件来压缩，或者os.listdi+os.path.isdir结合判断)
  • zipobj.namelist()：列出压缩包内容
  • zipobj.extractall(path=’.’)：解压压缩包内容
  • zipobj.close()：关闭压缩、解压对象
