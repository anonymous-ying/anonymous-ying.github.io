网络通信原理


基于OSI七层模型中的传输层

  • TCP：可靠(不代表安全)的传输协议、基于三次握手(SYN -> ACK/SYN -> ACK),相对于UDP效率低
  • UDP：不可靠的传输协议、面向无连接，只负责发送，不管是否到达，不基于握手模式，效率高于UDP

    Socket模块


    整体流程图

    监听、连接

    obj = socket.socket()：创建套接字对象 (C/S)
        参数一：socket.AF_INET[6] - IPv4[6]、AF_UNIX - unix
        参数二：socket.SOCK_STREAM - TCP | socket.SOCK_DGRAM - UDP
    obj.setblocking(0|1)：设置是否需要阻塞
        0：非阻塞
        1：阻塞
    obj.settimeout(time)：设置连接超时时间
    obj.bind(SOCKET)：申请绑定本地的一个端口进行监听 (S)
        SOCKET=(ip,port) - 本地监听的套接字
    obj.listen(NUM)：设置最大的监听请求数 (S)
        NUM - 默认为1,一般设置为5就够了
    conn,address = obj.accept() : 等待客户端连接 (S、阻塞会话)
    返回值：
        conn - 连接成功的会话对象
        address(ip,port) - 连接进来的套接字
    obj.connect(SOCKET)：主动去连接一个监听中的端口 (C)
        SOCKET=(ip,port) - 服务器端的套接字

    数据传输

    obj.send(data) - 向连接的另一方发送数据 (C/S)
    obj.recv(length) - 接收来自另一方的数据 (C/S、阻塞会话)

    BIF

        socket.gethostname()：返回当前主机名
        socket.gethostbyname(hostname)：返回参数(主机名)对应的IP
        socket.getservbyport(port[,protocol])：返回端口对应的协议类型
        socket.getservbyname(protocol[,])：返回协议类型对应的端口
        进制转换
            socket.inet_aton(ip)：将IP转换为32位的二进制格式(binary)
            socket.inet_ntoa(binary)：将binary转换为IP格式(十进制)
            转换为长字节(32-bit)
                socket.ntohl[htonl](data)
            转换为短字节(16-bit)
                sock.ntohs[htons](data)

实现异步

  • 模块：socketserver
        1、重构socketserver.BaseRequestHandler为自定义类
      2、将自定义的类但做参数传给[server = ]socketserver.ThreadingTCPServer((SOCKET), MyTCPHandler)
        3、启动实例的对象 - server.serve_forever()
import socketserver
class MyTCPHandler(scokerserver.BaseRequestHandler):
    def handle(self):
        """
        定义创建socketServer
        self.request -> conn对象
        """
        print "New Conn:", self.client_address
        data = self.request.recv(1024)
        print "CLient Says:", data
        self.request.send(data)
if __name__ == '__main__':
    HOST, PORT = "127.0.0.1", 8888
    server = socketserver.ThreadingTCPServer((HOST,PORT), MyTCPHandler)
    server.serve_forever()

端口扫描

  • 原理:使用connect()去尝试连接主机端口，如果端口没有开放便会触发拒绝连接(Connection refused)的异常

    简单测试脚本
#coding:utf-8
import socket

def main():
    # ports = range(1,1024)
    ports = [20,21,22,23,25,80,3306,3389]
    ip = '127.0.0.1'
    print "[*]Start Scan %s port" % ip
    for p in ports:
        c = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        conn_socket = (ip, p)
        try:
            c.connect(conn_socket)
            print "[+]%s is up" % p
        except Exception as e:
            print "[-]%s is down" % p
        finally:
            c.close()

if __name__ == '__main__':
    main()
  • 每次循环必需重新创建一个套接字对象，不然会出现Bad file descriptor异常
  • 套接字打开后记得close

交互聊天终端

代码示例


socket Server
#coding:utf-8
import socket

bind_ip = ('127.0.0.1', 8888)

def main():
    '''socketServer'''
    server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server.bind(bind_ip)
    server.listen(5)
    while True:
        print "[*] Waiting connect, Enter CTRL+C quit"
        try:
            conn, address = server.accept()
            print "[+] %s connecting" % address[0]
            conn.send("Welcome to connect in %s port" % bind_ip[1])
            while True:
                request = conn.recv(1024).strip()
                if request == "quit":break
                print "[%s:%s] recv: " % (address[0],address[1]), request
                server_input = raw_input(">>>")
                if server_input == 'q' or server_input == 'quit':
                    conn.send('quit')
                    break
                conn.send(server_input)
            conn.close()
        except KeyboardInterrupt as e:
            print "[*]Server will be quit!!!"
            break
    server.close()

if __name__ == '__main__':
    main()
  • 第一个循环：当一个会话断开时候，继续等待下一个会话
  • 第二个循环：会话数据交互循环，只要还有数据传输就一直处于循环阶段
  • recv会阻塞会话，让会话一直处于等待阶段，直到有数据发送过来
  • raw_input同样会阻塞会话，知道键入enter结束
  • 当需要停止会话的时候，只需要触发break跳出数据交互循环便可退出当前会话，等待下一个会话

    模拟SSH

  • 原理：将客户输视为终端执行的命令，通过命令执行模块(os.system|os.popen|subprocess)将命令的输出通过send发送给客户端

    SSHServer代码
#coding:utf-8
import socket
import subprocess

bind_ip = ('127.0.0.1', 8888)

def main():
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.bind(bind_ip)
    s.listen(5)

    while True:
        print "[*]Waiting connect, Enter CTRL+C quit..."
        #做一个CTRL+C的退出一场处理
        try:
            conn, address = s.accept()
            while True:
                #阻塞等待用户数据
                client_data = conn.recv(1024)
                #但客户端执行q的时候，没有数据过来就退出
                if not client_data:break
                print "RECV cmd:", client_data

                #执行客户端发送过来的命令
                cmd = client_data.strip()
                cmd_status = subprocess.call(cmd, shell=True)
                if cmd_status == 0:
                    cmd_call = subprocess.Popen(cmd, shell=True, stdout=subprocess.PIPE)
                    cmd_result = cmd_call.stdout.read()
                    #如果命令没有返回结果(cd)
                    if len(cmd_result) == 0:
                        cmd_result = "cmd execution has no output..."
                    #为了避免接收的长度有误，先将命令输出的长度以及确认标识发过去
                    ack_msg = "CMD_RESULT_SIZE|%s" % len(cmd_result)
                    conn.send(ack_msg)
                    #避免粘包 - 阻塞，等待客户端确认
                    client_ack = conn.recv(50)
                    if client_ack == "CLIENT_READY_TO_RECV":
                        conn.send(cmd_result)
                else:
                    conn.send("[-]Please check command")
        except KeyboardInterrupt as e:
            break
        finally:
            conn.close()

if __name__ == '__main__':
    main()

#coding:utf-8
import socket

conn_ip = ('127.0.0.1', 8888)

def main():
    c = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    c.connect(conn_ip)

    while True:
        #阻塞等待输出命令
        user_input = raw_input("cmd:").strip()
        if len(user_input) == 0:continue
        if user_input == 'q':break

        #发送命令到服务端
        c.send(user_input)
        #ack_msg = "CMD_RESULT_SIZE|%s" % len(cmd_result)
        server_ack_msg = c.recv(100)

        #确认命令是否正确
        if '[-]' in server_ack_msg:
            print server_ack_msg
        elif '|' in server_ack_msg:
            #确认发送过来的数据长度
            cmd_res_msg = server_ack_msg.split("|")
            if cmd_res_msg[0] == "CMD_RESULT_SIZE":
                cmd_res_size = int(cmd_res_msg[1])
                c.send("CLIENT_READY_TO_RECV")
            res = ''
            received_size = 0
            #如果文件过大，为了避免读死，每次只接收500,直到收完
            while received_size < cmd_res_size:
                #将已接收的长度跟数据长度比较，直到接收完
                server_reply = c.recv(500)
                received_size += len(server_reply)

                res += server_reply
            else:
                print res
                print "----------recv done---------"

    c.close()

if __name__ == '__main__':
    main()

  • 输入的是否为正确命令

  • 命令没有执行结果

  • 接收数据过大，recv为不定值，一次性接收数据不要过大
