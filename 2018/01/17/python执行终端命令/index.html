Python(Shell)执行终端命令模块

发表于 2018-01-17 | 分类于 Python
调用os | subprocess执行终端命令

pythonShell使用

  • %run file.py：执行python文件
  • who[s]：查看当前shell中的变量
  • reset：清空当前shell中的变量
  • dir(module)：查看模块可用函数、变量
  • type(value)：查看变量的类型
  • func?[help(func)]：查看函数帮助
  • 执行终端命令： !command - 可以将命令结果保存为变量


当调用执行命令模块的时候，生成一个子shell来执行命令，执行完成后再关闭子shell(子进程)

OS模块

os.system()

    格式：os.system(command)
    返回值：命令执行的状态码
        0：命令正常执行
        !=0：命令执行错误
    应用场景：只需要命令执行的结果，不用将命令的结果输出

os.popen()

    格式：os.popen(command)
    返回值：一个文件对象，命令将执行的结果写入一个临时文件，通过read()读取命令的执行结果
    应用场景：需要调用命令的执行结果

subprocess模块

subprocess.call()

    格式：subprocess.call(command,shell=True)
        参数一：command - 默认需要以列表的方式来拼接commad的参数
        参数二：shell=True - 不使用列表的方式，直接以终端方式执行
    返回值(同os.system)：命令执行的状态码

subprocess.Popen()

    格式：subprocess.Popen(command,shell=True,stdout[stdin|stderr])
        参数三：stdout[stdin|stderr] = subprocess.PIPE - 因为命令的执行实在子shell中执行的，如果需要将子shell(子进程)的数据传回父shell需要进行进程间通信，使用一个subprocess的PIPE管道函数将执行的结果返回给父shell
    返回值：subprocess.Popen对象
        读返回值：obj.stdout - 文件对象，通过read()读文件
