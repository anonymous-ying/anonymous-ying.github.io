Python脚本参数处理模块

发表于 2018-01-17 | 分类于 Python
Python脚本参数解析模块 - (sys、optparse、argparse)

sys

  • sys.argv：列表的形式返回外部参数，实现参数解析，程序外部向内部赋值
        列表第一个值为脚本自身文件名
        参数：sys.argv[1:]
        识别空格(可以间隔多个空格)，作为参数解析
  • sys.path：列表的形式返回Python模块的环境变量
        用于添加自定义模块的路径
  • sys.platform：字符串的形式返回系统类型
  • sys.getdefaultencoding()：字符串的形式返回系统当前的编码，默认ascii
  • sys.setdefaultencoding(ENCODING)：设置系统的默认编码
        必需先reload(sys)才可以执行该函数
  • sys.getfilesystemencoding()：字符串的形式返回文件系统使用的编码

optparse

1、实例化参数解析对象

parser = optparse.OptionParser(‘USE_HELP’[, usage=’USE_HELP’, version=’INFO’, description=’MSG’])
        %prog：当前脚本名称

USE_HELP = "usage: %prog -h <help doc> -a <all> --argument <argument>"

2、设置参数

parser.add_option([OPTION])
    默认使用-h/–help选先(–version,需先定义version)
OPTION

  • -：简写参数
        不支持长参数 -> -all
        参数可以连起来，也可以识别空格
  • --argument：全称参数 <=> dest=’argument’
        可以识别=号分割，也可以识别空格
        声明该选项相当与定义了dest选项的变量为argument
  • dest：声明存储参数的变量名
        被parse_args()的第一个返回值(option)解析对应
  • action：进行参数处理
        store：将对应参数存储到dest指定变量，默认
        store_true[false]：如果dest没对应参数将设置为true[false]
        append：以列表的形式将参数存储到dest
            多次指定对应参数才能达到添加多个值到列表的效果…
  • type：参数的值类型(限定传参时候的类型,而不是传入后的类型)
        单字符类型：int、str、float
        选择类型：choice
  • choices=[list]：当type为choice使用
  • default：设置该参数的默认值
  • help：该参数的使用帮助

3、解析参数

option, args = parse.parse_args()
    option：调用参数设置的dest
        字典形式 - {dest:argument}
    args：没被解析的命令行参数列表
        一个dest只能对应一个argument，如果是多个参数对应一个dest将会被视为没被解析的参数

4、赋值参数

        arg1 = option.DEST
        做参数不为空处理

if option.DEST == None:
    print parser.usage

示例

#coding:utf-8
import optparse

def main():
    parser = optparse.OptionParser('%prog -H <specify target host[s] -p <specify target port[s]> -m <specify attack module>')
    parser.add_option('-H', '--host', type='string', help='specify target ip[s]')
    parser.add_option('-p', '--port', type='int', default=80, help='specify target port[s]')
    parser.add_option('-m', '--module', type='choice', choices=['tcp', 'udp', 'syn'], help='specify attack module')
    options, args = parser.parse_args()

    if options.host == None and options.module == None:
        print parser.usage
        exit(0)

    print options, args
    host = options.host
    port = options.port
    module = options.module
    for host in args:
        print host, type(host), port, type(port), module, type(module)

if __name__ == '__main__':
    main()

argparse


类似optparse， 维护版optparse, 支持-的长参数以及一个dest传入多个参数(列表形式)
1、实例化解析对象
parser = argparse.ArgumentParser(usage=’USE_HELP’[, version=’INFO’, description=’MSG’])
2、参数设置
parser.add_argument([OPTION])
区别于optparse的OPTION

  • 指定type时候不需要单引号阔起(int, str, float)
  • choices=[list], 不需要声明类型为choice，但需要声明输入的类型
  • nargs：参数的个数(解决optparse的坑) - nargs=’? | * | +’
        ？：必需传一个参数
        *：0到多个参数
        +：1到多个参数

3、解析参数
args = parser.parse_args()
args：名称空间类型
通过args.DEST直接引用参数

4、赋值参数
arg1 = args.DEST
参数错误或者不想让参数为空提示帮助信息
if args.DEST == None:
parser.print_help()
exit(0)
示例

import argparse
def main():
    parser = argparse.ArgumentParser(usage='%(prog)s -H <specify target host> -p <specify target port[s]> -m <specify attack module>', description='menber the hosts and ports')
    parser.add_argument('-H', '--host', dest='host', type=str, default='127.0.0.1', help='specify the target host address(ip)')
    parser.add_argument('-p', '--port', type=str, nargs='+', help='specify the target host accept port')
    parser.add_argument('-m', '--module', type=str, choices=['tcp','udp','syn'], help='specify the target host accept port')
    args = parser.parse_args()

    if args.port == None or args.module == None:
        parser.print_help()
        exit(1)

    print args
    print args.host
    print args.port
    print args.module

if __name__ == '__main__':
    main()
