Python模块基础及常用模块

模块基础

何为模块：大牛将代码功能封装成包为我们提供简易的操作接口，可以直接给我们使用，而无需关系模块功能是怎么实现的，就像一台电脑，每个硬件就可以看作一个模块，我们只需要将模块调用拼接就可以成为我们使用的主机

  • 模块安装方式
        1、使用别人打包的压缩文件
            执行压缩文件中的setup.py文件
        python setup.py install
        2、whl格式安装包
            安装wheel模块(pip install wheel)，无需重复安装
            执行：pip install file.whl
        3、使用pip
            一般python自带，有些需要修改环境变量才能使用
            安装模块：pip install MODULENAME
            卸载模块：pip uninstall MODULENAME
            查看已经安装模块：pip list all
            修改pip源：
                Linux - ~/.pip/pip.conf
                Windows - %appdata%\pip\pip.ini

常用模块列表

  • sys：操作终端，或者设置python环境
  • os：操作文件路径或者文件
  • shutil：操作文件
  • zipfile/tarfile：文件压缩/解压
  • optparse/argparse：做参数解析
  • getpass：输入隐藏，如输入password
  • threading：多线程
  • multiprocessing：多进程
  • hashlib：数据加密
  • time：时间设置
  • re：正则表达式，做字符串过滤
    网络编程
  • subprocess：执行终端命令
  • psutil：获取系统信息
  • IPy：解析IP地址
  • dns.resolver：做DNS解析
  • pycurl：做web质量检测,类是curl命令
  • socket：网络通信套接字
  • struct：将python类型转换为C struct类型，将数据打包成二进制流进行网络传输
  • scapy：对流量数据包[pcap]解析(类wireshark)
  • pexpect：实现SSH、FTP、PASSWD、TELNET等自动交互程序
  • paramiko/pexpect.ssh：实现SSH功能
  • fabric：类似SSH，远程执行命令
  • ftplib：python的FTP接口
  • smtplib：实现邮件发送功能
  • ConfigParser：做配置文件解析
  • logging：自定义程序日志
    爬虫
  • request/urllib[2]：web页面请求 - 静态页面
  • selenium：模拟浏览器进行web页面请求 - 动态加载页面
  • urlparse：做url解析，分解url为多个部件
  • cookielib：做web-cookie存储插件
  • lxml/bs4：做web页面元素筛选
  • csv：excel格式存储数据
  • mongodb：mongodb的python-API
  • MySQLdb：mysql的python-API
  • sqlite3：sqlite的python-API
  • json/pickle/shelve：存储内存数据
    数据分析
  • match：提供基本的数学函数
  • numpy：提供数组或者矩阵处理
  • scipy：基于numpy提供科学计算(概率统计、曲线拟合、积分、线代…)
  • pandas：提供序列和数据框架，针对结构化数据
  • matplotlib/gdchart：根据数据制图
    高阶
  • nltk：自然语言处理
  • Scikit-learn：数据挖掘分析
  • Theano：高效定义、优化和评估数学表达式
